Java Array Input Validator
 Project Overview
This project is a Java console-based application that demonstrates fixed-size array allocation with runtime user input validation.
The program allows users to:

Define the array size dynamically
Enter integer values
Validate all inputs to prevent non-integer entries
Handle exceptions gracefully

Developed using IntelliJ IDEA as part of academic coursework in Data Structures and Programming.

 Technologies & Tools

Java (JDK 17+)
IntelliJ IDEA
Scanner Class
Exception Handling (InputMismatchException)


 Core Functionalities
 Runtime array size allocation
 Integer-only input enforcement
 Exception handling with retry logic
 Defensive programming principles

 How to Run (IntelliJ)

Clone the repository:

   git clone https://github.com/Gilbert86/java-array-input-validator.git

Open IntelliJ IDEA
Click Open → Select project folder
Ensure JDK 17+ is configured
Run ArrayInput.java
